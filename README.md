# Reich Studio

Marketing- und KI-Automatisierungs-Website für Reich Studio. Gebaut mit Next.js (App Router) und Tailwind CSS v4.

## Entwicklung

```bash
npm install
npm run dev
```

Öffne [http://localhost:3000](http://localhost:3000).

## Struktur

- `src/app` — Layout, globale Styles, Metadata
- `src/components` — Header, Hero, Problem, Services, About, Contact, Footer, Reveal (Scroll-Animation)

## Deploy

Deployed auf [Vercel](https://vercel.com).
